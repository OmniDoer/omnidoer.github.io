Handex

A manual fallback for Codex-style work: copy a prompt into any web LLM, review one tool command, run it locally, and return the result.

View on GitHub How it works

Copy -> Review -> Run -> Return

Handex keeps the agent loop usable when automated quota is unavailable. The model still reasons. The human still approves. Local tools still do the work.

CopySend a compact project prompt to a web LLM.
ReviewPaste back one JSON tool command and inspect the exact operation.
RunExecute local shell, file, patch, Git, skill, vault, or OmniDoer bridge tools.
ReturnCopy the tool result prompt back so the model can continue.

Built for fallback continuity

SkillsReads Codex-style SKILL.md instructions and referenced files through progressive disclosure.
VaultLists credential metadata and runs reviewed vault-backed actions without putting secrets in chat.
ContextKeeps workspace context, plans, uploads, tool history, summaries, and transcripts durable.
