# Windows 11 Control Client

MVP Windows support is the HTML5/PWA Control Client in Edge or Chrome.

Future native Windows client plan:

- WebView2 or Tauri shell.
- Windows Hello integration.
- DPAPI or Windows Credential Manager for local keys.
- Native notifications.
- E2EE pairing with Broker.
- Unified approval, credential, challenge, and takeover UI.
- Windows Hello platform authenticator integration.
- Low-latency remote browser takeover.
